# 클린 코드

자바에만 적용되는 특성이 많아서 그런가 이해하기가 조금 어려웠다. 해당 내용은 구름 EDU에서 클린 코드 자바스크립트 번역본 내용을 참고해서 작성한 것이다.
클린 코드 책을 한번 다 읽고 다시 돌아와서 읽어봐야겠다.

## getter와 setter를 사용해라.

- 내부용 API를 캡슐화 할 수 있다.
- set 할 떄 검증 로직을 추가할 수 있다.
- 로그를 찾거나 에러 처리하기가 쉽다.
